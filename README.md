# ForteBank Guard — MVP Fraud Detection

Минимально рабочий прототип системы поиска мошенничества.  
Полностью готово для запуска локально!

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Генерация данных

```bash
python data_gen.py
```

## 3. Обучение модели

```bash
python train_model.py
```

## 4. Запуск Streamlit-приложения

```bash
streamlit run app.py
```
